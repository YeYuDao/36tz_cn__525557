<br/></br>### 课程介绍:
Ansible是一种自动化的运维工具，基于Python开发，它集合了众多运维工具（比如puppet、chef、func等）的优点，能够实现批量操作。但其实Ansible是基于模块工作的，本身没有批量部署的能力。真正具有批量部署的是ansible所运行的模块，Ansible只是提供一种框架。
